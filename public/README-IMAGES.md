# Инструкция по созданию изображений для SEO

## Требуемые изображения

Для полной SEO оптимизации необходимо создать следующие изображения и разместить их в этой папке (`public/`):

### 1. og-image.png (1200x630px)
**Назначение:** Open Graph изображение для всех страниц сайта

**Требования:**
- Размер: 1200x630 пикселей
- Формат: PNG или JPG
- Содержание:
  - Логотип компании "Выкуп авто"
  - Название компании
  - Ключевой текст: "Выкуп автомобилей в Москве и МО"
  - Контактные телефоны (опционально)
- Используется в: `lib/seo.ts` как `defaultImage`

### 2. icon-192.png (192x192px)
**Назначение:** Иконка для PWA (малый размер)

**Требования:**
- Размер: 192x192 пикселей
- Формат: PNG
- Квадратная иконка с логотипом компании
- Используется в: `app/manifest.ts`

### 3. icon-512.png (512x512px)
**Назначение:** Иконка для PWA (большой размер)

**Требования:**
- Размер: 512x512 пикселей
- Формат: PNG
- Квадратная иконка с логотипом компании
- Используется в: `app/manifest.ts`

### 4. logo.png
**Назначение:** Логотип компании для Schema.org разметки

**Требования:**
- Рекомендуемый размер: минимум 600x60 пикселей
- Формат: PNG (с прозрачным фоном) или JPG
- Логотип компании
- Используется в: `app/layout.tsx` в Organization и LocalBusiness Schema

## Инструменты для создания

Рекомендуемые инструменты:
- **Canva** (https://www.canva.com) - бесплатный онлайн редактор
- **Figma** (https://www.figma.com) - профессиональный дизайн-инструмент
- **Photoshop** / **GIMP** - графические редакторы
- **Online Image Resizer** - для изменения размеров существующих изображений

## Проверка после создания

После создания изображений проверьте:
1. Все файлы находятся в папке `public/`
2. Размеры файлов соответствуют требованиям
3. Изображения оптимизированы (сжаты без потери качества)
4. Файлы доступны по URL: `https://mos-avto-alyans.ru/[имя-файла]`

## Оптимизация изображений

Рекомендуется оптимизировать изображения перед размещением:
- Используйте инструменты сжатия (TinyPNG, ImageOptim)
- Для PNG: сохраняйте с прозрачностью только если необходимо
- Для JPG: используйте качество 80-90%
- Проверьте размер файлов (целевой размер: < 200KB для og-image, < 50KB для иконок)
